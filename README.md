🧠 Concept

Attendance Calculator helps students avoid shortage by giving instant clarity on:

Where they stand
How much margin they have
What actions to take

👉 Enter data → Get insights
👉 Plan attendance → Avoid detention

Smart tracking → Better academic control

⚙️ Logic Behind
Attendance % = (Attended / Total) × 100
If above required → calculate bunkable classes
If below required → calculate classes needed
